# Custom Hooks

Reusable React hooks for managing app state and logic.

## Files

- **useRestaurants.js** - Main hook for restaurant data management
  - Handles fetching restaurants from Firebase
  - Manages restaurant state
  - Provides functions to save new restaurants and add ratings
  - Handles error logging and display
  - Manages loading states

This hook encapsulates all Firebase/Firestore operations and keeps the main App component clean.

# Map Components

Components related to the Leaflet map display.

## Files

- **RestaurantMap.jsx** - Main map component that displays all restaurant markers
- **MapEvents.jsx** - Helper component to access and pass the map instance to parent
- **createPixelPin.js** - Utility function that creates custom pixel-art map pins with emojis based on cuisine type

The map uses Leaflet and react-leaflet libraries to display an interactive map with custom markers for each restaurant.

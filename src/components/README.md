# Components

This folder contains all the React components used in the app.

## Structure

- **Map/** - Map-related components (map display, markers, pins)
- **Modals/** - All modal dialogs (search, add restaurant, ratings, preview)
- **ErrorToast.jsx** - Error notification toast component
- **RestaurantSearch.jsx** - Google Places search component with autocomplete
- **SidePanel.jsx** - Right-side panel showing restaurant details
- **ControlPanel.jsx** - Bottom-left control buttons (Add, Random picker)

Each component is self-contained and receives its data/functions via props.
